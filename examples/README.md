# Example Data Files

This directory contains example data files to help you understand the expected format for each stage of the analysis.

## File Formats

All example files are provided as CSV format for easy viewing and editing. The application also accepts XLSX files.

## Creating Your Own Data

You can use these examples as templates:

1. **Sales Data**: Copy the structure of `amazon_sales_example.csv` or `tiktok_sales_example.csv`
2. **Comments**: Copy the structure of `tiktok_comments_example.csv`
3. **Reviews**: Copy the structure of `amazon_reviews_example.csv` or `tiktok_reviews_example.csv`

## Notes

- Column names are case-insensitive
- The app will automatically detect columns based on common naming patterns
- Optional columns can be omitted
- Use your actual data with real values for best results
